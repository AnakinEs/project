Несколько одинаковых функций

Если создать несколько функций Function Declaration с одними и теми же именами, всегда будет выполнена последняя функция.

Но если создать несколько функция Function Expression с одними и теми же именами, все функции будут взаимонезависимы, они все будут выполнены.

Такое поведение объесняется тем, что Function Declaration создаются интерпретатором до выполнения кода, во время инициализации, а Function Expression создается во время выполнения выражения, как только интерпретатор дойдет до строчки с функцией в коде.

Нужная последняя функция Function Declaration будет выбрана интерпретатором во время инициализации. Function Expression создается по порядку, как только дойдет до строчки кода. Будет создана и выполена одна функция, далее с тем же именем создана и выполена другая, и так далее. По сути одна и та же переменная перезаписывается на другую функцию после ее создания и завершения.
