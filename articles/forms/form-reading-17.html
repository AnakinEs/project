Элемент <textarea>

<textarea> элемент определяет многострочное поле ввода (область текста):

<!DOCTYPE html>
<html>
<body>

<h2>Textarea</h2>
<p>The textarea element defines a multi-line input field.</p>

<form action="/action_page.php">
  <textarea name="message" rows="10" cols="30">The cat was playing in the garden.</textarea>
  <br>
  <input type="submit">
</form>

</body>
</html>

Атрибут rows указывает видимое число строк в текстовой области.

Атрибут cols определяет видимую ширину текстовой области.

Можно также определить размер текстовой области с помощью CSS:

<!DOCTYPE html>
<html>
<body>

<h2>Styling Textarea</h2>
<p>Use CSS to change the size of the textarea:</p>

<form action="/action_page.php">
  <textarea name="message" style="width:200px; height:600px;">The cat was playing in the garden.</textarea>
  <br>
  <input type="submit">
</form>

</body>
</html>
